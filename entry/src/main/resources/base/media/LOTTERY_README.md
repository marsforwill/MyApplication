# 抽签功能图片资源说明

## 需要的图片资源

### 签筒图片
文件名：`lottery_tube.png`
位置：`entry/src/main/resources/base/media/`

**说明**：
- 用于显示在抽签页面的签筒图片
- 建议尺寸：400x500 像素或更大
- 格式：PNG（支持透明背景）
- 如果没有此图片，系统会自动使用卡牌背面图片作为占位

**临时解决方案**：
如果暂时没有签筒图片，可以：
1. 使用现有的 `card_back.png` 作为临时占位
2. 或者找一张中国风的竹签筒、木质签筒图片
3. 在线搜索关键词："签筒 PNG"、"观音灵签 签筒"

## 功能说明

抽签功能已经实现，包括：
- ✅ 100支观音灵签数据（示例包含20支完整签文）
- ✅ 摇签动画效果
- ✅ 签文展示（签号、等级、诗句、含义）
- ✅ 详细解签页面（包含姻缘、事业、财运、健康运势）
- ✅ 从首页"观音灵签"按钮进入

## 扩展建议

如需完整的100签数据，可以：
1. 参考传统观音灵签书籍
2. 在线搜索"观音灵签100签全解"
3. 按照 `LotteryStick.ets` 中的数据格式继续添加

当前已提供20支签文作为示例，可以正常运行和测试功能。

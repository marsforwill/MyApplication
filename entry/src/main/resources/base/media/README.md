### 图片规格建议

- **格式**：PNG（支持透明背景）或 JPG
- **尺寸**：建议 400x640px 或更高（保持 5:8 的宽高比）
- **文件大小**：每张图片建议不超过 500KB
- **风格**：神秘、优雅、符合塔罗牌传统风格

### 临时方案

如果暂时没有塔罗牌图片，可以使用以下方法：

1. 使用占位图片：创建简单的纯色或渐变图片作为占位符
2. 使用在线资源：从公开的塔罗牌图片库下载（注意版权）
3. 使用AI生成：使用AI工具生成塔罗牌风格的图片

### 图片命名规范

所有图片文件名必须严格按照上述命名规则，否则应用将无法正确加载图片资源。
